# Week 1: Introduction to User Interface Design

Welcome to the User Interface Design (UID) course! This course will teach you how to design a mobile application from begining to end. While we cover some of the essentials of the UX process, our main focus will be design principles, platform specifications and advanced use of the Figma design tool. The UX process will be explored more deeply in level 3 MAD9034. Our class meets once a week for three hours to learn Design processes and apply practical knowledge together in the lab. To make the most of this course, it's crucial to attend the class as there will be guided practical learning and in class exercises. Active participation and attendence is essential to your success in the course.

Reading and watching the content provided each week before class will allow you to be more prepared and to better grasp the various concepts and techniques.

# Learning Objectives

By the end of this week, you will:

  • Get an overview of the content & expectations of MAD9020 "User Interface Design course".
  • Usability Heuristics: understanding & applying them in interface design.
  • Practice your Knowledge: conduct a heuristic evaluation of a mobile application and a website.

# Usability Heuristics

10 usability heuristics for interface design (opens new window)

  1. Visibility of System Status
  2. Match Between the System and the Real World
  3. User Control and Freedom
  4. Consistency and Standards
  5. Error Prevention
  6. Recognition Rather than Recall
  7. Flexibility and Efficiency of Use
  8. Aesthetic and Minimalist Design
  9. Help Users Recognize, Diagnose, and Recover from Errors
  10. Help and Documentation
