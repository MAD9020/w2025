# Week 2 - Layouts & Grids

# Step 1: Create a cover for your Design system file

In your previous assignement you created a Mad9020 folder, within this folder, create a new file and call it:"YourName-Design-System". Follow the video below to understand the basics of keeping your design files clean and well organized and to create a cover as the first page for your file. The cover must have your photo on the right side and the information on the left.



# Step 2: Create your own responsive Grids styles

Create a second page in your file, call it grids. You can follow in class instructions or use the 8pt grid video below to acheive this step.Create 3 frames for desktop, tablet and mobile. Create responsive grids for all 3 of them:

  1. The column grid to allow horizontal alignment. Number of columns changes for different devices: 12, 8, 4.
  2. The row grid to allow for vertical alignment, make sure the difference between rows is 8px with gutter of 0px.
  3. Create a reusable grid style that combines columns and rows for each device. The 3 grid styles you create should be named for desktop, tablet and mobile.

# Step 3: Practice auto - layout

Follow the Figma video below to practice autolayout and create the objects created.1

  1. Button + icon
  2. Menu
  3. Dialog

# Step 4: Practice alignment and balance

Arrange the cards along with other shapes like rectangles for footer and header on your frames following the UI design principles we learned to create a cohesive well balanced design. Do not bother adding colours or styles at this stage, let your main focus be on alignment, proximity, balance, similarity...

# Things to consider:

  1. How will the design layout change from one device size to another?
  2. How to balance elements on your page?
