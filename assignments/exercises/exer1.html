# Week 1 - Heuristic Evaluation

# Details

Select from the 2 options below a combination of a mobile app and website for which you will conduct a heuristic evaluation following the learning from today's class. The 10 usability heuristics can be accessed here (opens new window).

Option 1 Option 2
1.Amazon mobile app 1.Expedia Mobile app
2.Diana wix portfolio website (opens new window) 2.Saloni wix portfolio website (opens new window)

# Requirements:

  • Go through the app and website and specify which usability heuristics are applied or broken. List a minimum of 3 heuristics for each product.
  • Use screen shots to showcase each point you are trying to make. For ex: a screenshot could show missing navigation where the user cannot exit a page or the opposite with visible well designed navigation.

# Deliverable for this exercise:

Sharing your work with the teacher

  • Create a Mad9020 folder in your Figma workspace. Make sure you are using Figma education and not the free account. Share the whole folder with gereign@algonquincollege.com. Add my email with edit privilege. if you do this on the level of the folder you will not have to repeat it every time.
  • Create a new figma file and give it a name in this format: your-name-Heuristic Evaluation.
  • Add your heuristic evaluation of both app and website with screenshots and observations. Always name and refer to the specific usability heuristic and how it was followed or not followed. Make sure to specify implications on the user in, how they are affected in their workflow.
  • Design Bonus: Make you findings look good and well organized.

# Submission

  • Open Brightspace LMS and go to the Activities > Assignments page.
  • Go to the Week 1 - Heuristic Evaluation assignment.
  • Submit a link to your figma file.
