# MAD9020 Deliverables

Every week there will be work for you to complete inside or outside of class. There will be 8 exercises (50%), Midterm (20%), Final project (30%).

Week 1 - Heuristic Evaluation

Following Wednesday- midnight

In Week 1, we delve into understanding usabillity Heuristic and conducting an evaluation for a website and mobile app.

Following Wednesday - midnight

Week 3 - Colour System

Following Wednesday - midnight

Following Wednesday - midnight

In Week 11, we create a high-fidelity prototype that includes Conditional logic in Figma with variables, variants and Modes. This assignment aims to help students understand the figma logic to create realistic interactive prototypes for usability testing.
